# Compound Deception (Ansari) - ansari100 Seed Dataset

This dataset is a 100-entry subset derived from the public GPTZero NeurIPS 2025 hallucination table referenced by Ansari (2026).

- Source sheet: https://docs.google.com/spreadsheets/d/14hIuCK7HCnGhdCuxfwp1wIYbbHMn-K3FNvY9NgWGW0Y/edit?usp=sharing
- Retrieved: 2026-04-09
- Included entries: first 100 rows from the 100-row public table snapshot

Files:
- ansari100.bib: BibTeX-formatted @misc entries with extracted title/author/year when possible
- metadata.json: per-entry ground truth and provenance metadata
- gptzero_neurips_100_source.csv: source snapshot for reproducibility
